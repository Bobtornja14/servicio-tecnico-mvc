INSTRUCCIONES DE INSTALACIÓN

1. Extraiga el archivo ZIP
2. Ejecute install.bat (esto instalará todo lo necesario)
3. Una vez finalizada la instalación, ejecute run.bat para iniciar la aplicación
4. Abra su navegador en: http://localhost:5001

Requisitos previos:
- Python 3.8 o superior instalado
- Conexión a internet para descargar dependencias
